# Deinococcus radiodurans 文献查询系统 — 项目架构说明

## 一、项目概述

本项目是一个面向耐辐射奇球菌（*Deinococcus radiodurans*）研究领域的学术文献检索 Web 应用。整合了从 PubMed 获取的 1,344 篇规范化文献数据，支持通过**序号、标题关键词、作者名或 PMID** 进行多维度检索。

**核心特征：**
- 无需后端服务器即可运行（双击 `index.html` 或 `python main.py` 一键启动）
- 全部数据来自 PubMed API 的标准化字段
- 支持序号/关键词/作者/PMID 搜索
- **V8 增强匹配引擎**：前缀模糊匹配 + 期刊交叉验证 + 年份精确匹配 + 9因子综合评分
- 文献详情含标题、作者、期刊、年份、PMID、DOI、PMCID、摘要、MeSH Terms

## 三、V8 搜索算法核心逻辑

### 1. 查询清洗（cleanQuery）

从查询字符串中提取三部分信息：
- **标题词**：用于标题匹配的核心词汇
- **期刊词**：Source~SO 之间提取的期刊缩写
- **年份**：19xx/20xx 四位年份

### 2. 匹配策略（三阶段）

```
Phase 1: 严格整词交集  →  \bword\b 整词匹配，逐词过滤
Phase 2: 前缀模糊交集  →  \bword\w*\b 前缀匹配（处理截断词）
Phase 3: 综合评分排序  →  9因子评分，分差≥15自动唯一化
```

### 3. 综合评分维度（0~180+）

| 维度 | 最高分 | 说明 |
|------|--------|------|
| 匹配率 | 50 | 查询词匹配比例 |
| 位置分 | 25 | 匹配词集中在标题前半 |
| 连续子串 | 25 | 查询词完整连续出现 |
| 大小写 | 5 | 大小写精确吻合 |
| 词序一致性 | 30 | LIS 最长递增子序列 |
| 连续短语 | 40 | 相邻查询词在标题中也相邻 |
| 字符覆盖率 | 20 | 匹配字符占标题比例 |
| 期刊匹配 | 40 | Source~SO 间期刊名交叉验证 |
| 年份匹配 | 25 | 19xx/20xx 与数据库 year 精确匹配 |

### 4. Fallback 链

1. 严格整词交集
2. 前缀模糊交集
3. 综合评分排序
4. 子串匹配
5. 作者搜索
6. PMID 精确匹配
7. 摘要全文匹配

## 四、数据库字段说明

每条文献记录包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `number` | string | 文献序号（1-1344） |
| `pmid` | string | PubMed ID |
| `title` | string | 文献标题 |
| `authors` | string | 作者列表 |
| `journal` | string | 期刊名称 |
| `year` | int | 发表年份 |
| `doi` | string | DOI |
| `pmcid` | string | PMC ID |
| `abstract` | string | 摘要全文 |
| `mesh_terms` | array | MeSH 主题词 |
| `pub_types` | array | 出版物类型 |
